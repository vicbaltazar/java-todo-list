# To-do list em Java (console)

Pequena aplicação de linha de comando para gerenciar uma lista de tarefas.  
Projeto de estudo para praticar Java básico, `ArrayList`, `Scanner` e estruturas de repetição.

## Funcionalidades

- Listar tarefas atuais
- Adicionar novas tarefas
- Remover tarefas pelo número
- Sair do programa pelo menu

## Tecnologias

- Java 17
- IntelliJ IDEA
